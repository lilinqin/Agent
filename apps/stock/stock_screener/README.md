# A股选股器 📈

基于 **AKShare** 的免费A股选股脚本，筛选同时满足以下条件的股票：

- 📏 **接近年线（MA250）**：当前价格在250日均线 ±5% 以内
- 📉 **KDJ 超卖**：J值 < 13（表明短期严重超卖，有反弹预期）

## 安装依赖

```bash
pip install akshare pandas tqdm
```

## 运行

```bash
cd stock_screener
python stock_screener.py
```

## 输出示例

```
【银行】(3 只)
  代码       名称         现价    年线(MA250)  偏离%      K      D      J
  --------------------------------------------------------------------
  600036   招商银行     38.50       37.80   ▲1.85%   8.2    9.5    5.6
  601328   交通银行     5.82        5.65   ▲3.01%  10.1   11.2    7.8
  ...

结果已保存到：result.csv
```

## 参数说明（可在脚本顶部修改）

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `MA_PERIOD` | 250 | 年线周期（交易日数） |
| `MA_THRESHOLD` | 0.05 | 接近年线的偏离阈值（5% = ±5%） |
| `KDJ_J_MAX` | 13 | J值上限，小于此值才保留 |
| `KDJ_PERIOD` | 9 | KDJ计算周期（标准9日） |
| `HISTORY_DAYS` | 350 | 拉取历史K线天数，需 > MA_PERIOD |
| `SLEEP_BETWEEN` | 0.3 | 请求间隔（秒），防止被限流 |

## 注意事项

- 全市场扫描约 5000 只股票，完整运行需要 **30-60 分钟**
- 数据来源东方财富，不需要注册/Token，完全免费
- 建议在收盘后（15:30后）运行，数据最准确
- 结果仅供参考，不构成投资建议

## 数据来源

- 行业分类：东方财富行业板块
- K线数据：AKShare `stock_zh_a_hist`（前复权）
- KDJ：标准9日KDJ，自行计算（RSV → K=EWM(1/3) → D=EWM(1/3) → J=3K-2D）
